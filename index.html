HTML

  • The head element contains info about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space wew want around the content inside an element.
  • if using outside CSS must create link in head to have the designs carry over from .css file.

Git

  • git status: checks what branch we are currently in
  • git checkout -b branch-name: creates a new branch

  • Control flow is the order in which a compiter executes code in a script

  • After we will "call" the functions into action
